<h3 align="center"><font color="blue">Server Only</h3></font>
<p align="center">
<b>apt update && apt-get -y install git && git clone https://github.com/shadow046/openvpndeb && cd openvpndeb && chmod +x openvpn.sh && ./openvpn.sh</b>
